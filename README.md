# Exploratory Data Analysis (EDA) Project

## Objective
Analyze student performance data to uncover trends, correlations, and factors affecting academic success.

## Dataset
Student Performance Dataset

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
- Statistical Summary
- GPA Distribution Analysis
- Correlation Analysis
- Study Time vs GPA
- Absences vs GPA
- Parental Support vs GPA
- Sports Participation Analysis

## Key Findings
- Absences have a strong negative impact on GPA.
- Study time positively influences academic performance.
- Parental support improves student outcomes.
- Sports participation shows a slight positive effect on GPA.
